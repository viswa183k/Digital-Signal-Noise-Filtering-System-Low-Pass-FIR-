# Digital Signal Noise Filtering System (FIR)

This project implements a digital low-pass FIR filter to remove noise from analog signals. The system samples input signals, applies filter coefficients, and outputs a smoothed signal, improving signal clarity for communication and embedded applications.

## Features
- FIR-based digital filtering  
- Reduces high-frequency noise  
- Real-time signal processing  
- Improves signal integrity  

## Hardware Required
- Arduino board  
- Analog signal source  
- Output device (LED, motor, or oscilloscope)  

## Applications
- Communication systems  
- Sensor signal processing  
- Embedded DSP applications
